# Agent Context

This repository uses a structured documentation system for AI context.

**Start here:** [docs/README.md](docs/README.md)

## Quick Reference

| Need | Location |
|------|----------|
| Understand a feature | `docs/features/` |
| Check architectural decisions | `docs/adr/` |
| See current work plans | `docs/plans/active/` |
| Track work items | `docs/work/` |

## Working on This Repo

1. **Before starting work** - Check `docs/plans/active/` for existing plans
2. **New feature work** - Create a plan in `docs/plans/active/` first
3. **During implementation** - Update the plan's Progress Log
4. **After completion** - Update or create feature docs, move plan to `completed/`
